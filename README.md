# Profile App

A Flutter + Firebase based profile management application that allows users to securely log in using mobile OTP authentication and manage their personal profile details.

---

## Features

* Mobile OTP Authentication using Firebase Auth
* User Profile Management
* Edit Profile Information
* Upload and Update Profile Photo
* Educational Details Section
* Professional Details Section
* Phone Number Update
* Firebase Storage Integration
* Responsive UI Design
* Generate Profile Image Card

---

## Tech Stack

### Frontend

* Flutter
* Dart

### Backend & Services

* Firebase Authentication
* Firebase Storage

## Getting Started

### Prerequisites

* Flutter SDK
* Firebase Project
* Android Studio / VS Code
* FVM (Flutter Version Management)

---

## Installation

Clone the repository:

```bash
git clone https://github.com/22A81A43C9/profile-app.git
```

Navigate to project folder:

```bash
cd profile-app
```

Install dependencies:

```bash
fvm flutter pub get
```

Run the project:

```bash
fvm flutter run
```
